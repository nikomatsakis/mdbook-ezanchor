# mdbook-ezanchor

[![CI](https://github.com/nikomatsakis/mdbook-ezanchor/actions/workflows/ci.yml/badge.svg)](https://github.com/nikomatsakis/mdbook-ezanchor/actions/workflows/ci.yml)
[![crates.io](https://img.shields.io/crates/v/mdbook-ezanchor.svg)](https://crates.io/crates/mdbook-ezanchor)
[![docs.rs](https://docs.rs/mdbook-ezanchor/badge.svg)](https://docs.rs/mdbook-ezanchor)

An mdbook preprocessor that expands `{anchor}` references into source code
snippets with GitHub links. Mark regions in your source with `// ANCHOR: name`
comments and reference them from your book — no more stale copy-pasted code.
